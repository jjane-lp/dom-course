Adding JavaScript to HTML

Method 1: Script in <head>

JavaScript defined in the head section, available throughout the page.

<script>
  function headFunction() {
    alert('Hello from head!');
  }
</script>

Method 2: Inline JavaScript

JavaScript directly in HTML attributes like onclick, onload, etc.

<button onclick="alert('Inline JavaScript!')">Click Me</button>

Method 3: Script at End of Body

JavaScript at the bottom of the page - runs after HTML is loaded.

<script>
  // This script runs after the HTML is parsed
  document.getElementById('myButton').onclick = function() {
    // Do something
  };
</script>

Method 4: External JavaScript File

Link to external .js file (simulated here with internal script).

<script src="script.js"></script>
